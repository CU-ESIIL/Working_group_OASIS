# Outputs Image Slot

This folder is reserved for an outputs-related image used elsewhere on the site.

To change it, delete the current image and add one new `.png`, `.jpg`, `.jpeg`, `.webp`, or `.svg` file. The website will use the image in this folder automatically if a page references the `outputs` slot.

Use one image only. Figures, dashboards, maps, reports, or publication-ready visuals work well here.
