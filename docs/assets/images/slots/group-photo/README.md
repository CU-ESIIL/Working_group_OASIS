# Group Photo Image Slot

This folder controls the image used for group identity, team photos, or collaboration visuals.

Style guidance:

- A real group photo is appropriate here.
- If you use a placeholder or illustration, it can be more realistic than the square button images.
- Keep it welcoming and uncluttered.

How to replace it:

1. Delete the current image file.
2. Upload one new `.png`, `.jpg`, `.jpeg`, `.webp`, or `.svg` file.
3. Run `python scripts/generate_image_slots.py`.
4. Commit the image and regenerated files.

Use one image only.
