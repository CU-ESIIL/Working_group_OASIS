# Website Side Image Slot

This folder controls the image used for the homepage section about sharing the science on the website.

To change it, delete the current image and add one new `.png`, `.jpg`, `.jpeg`, `.webp`, or `.svg` file. The website will use the image in this folder automatically, regardless of filename.

Use one image only. Images that show summaries, maps, figures, dashboards, or community-facing communication work best.
